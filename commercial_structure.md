## 1. KAM — дилерская сеть

**Head:** Иванов Дильшод  
**Цель:** привлечение дилеров на классифайд, удержание, рост выручки с партнёрской сети  
**Owner Retention:** Иванов (крупных — лично, мелких/средних — через Support)

**Команда:**

| ФИО | Должность | CR |
|---|---|---|
| Иванов Дильшод | Head KAM | **0.59 🚨** |
| Мирзахмедов Батыр | Старший менеджер KAM | 0.92 |
| Норимов Зафар | Менеджер KAM | 1.67 |

**KPI:**

- Кол-во активных дилеров
- Выручка с партнёрской сети
- Retention % (доля удержанных клиентов) / Churn (доля оттока)
- DSO (Days Sales Outstanding) — срок погашения дебиторки

**Зоны ответственности Head:**

1. Стратегия развития партнёрской сети (дилеры, автосалоны)
2. Планирование и распределение портфеля между командой
3. Переговоры с крупными / стратегическими дилерами
4. Ценообразование на пакеты размещения
5. Передача обратной связи от дилеров в продукт
6. Отчётность перед коммерческим директором
7. Найм и развитие команды KAM

**Модель handoff с Support:**

Иванов приводит дилера → крупных ведёт лично → мелких/средних передаёт в Support для операционного обслуживания.  
Retention KPI = на Иванове (он owner).  
Support = исполнитель операционки (FRT, CSAT).

## Глоссарий

| Аббревиатура | Расшифровка |
|---|---|
| CR | Compa-Ratio — отношение зарплаты к рыночному бенчу |
| FRT | First Response Time — время первого ответа |
| TTR | Time To Resolution — время полного решения |
| CSAT | Customer Satisfaction Score — оценка клиента |
| FCR | First Contact Resolution — % решённых с первого обращения |
| SLA | Service Level Agreement — % соблюдения сроков |
| DSO | Days Sales Outstanding — срок погашения дебиторки |
| Retention % | Доля удержанных клиентов за период |
| Churn rate | Доля отточных клиентов |
| LTV | Lifetime Value — выручка с клиента за всё время |
| ARPU | Average Revenue Per User — средняя выручка на клиента |
| NPS | Net Promoter Score — индекс лояльности |
| KAM | Key Account Management — управление ключевыми клиентами |
| КПБ | Капиталбанк |
| CS | Customer Success — функция удержания/развития клиента |
| ОКК | Отдел контроля качества |
